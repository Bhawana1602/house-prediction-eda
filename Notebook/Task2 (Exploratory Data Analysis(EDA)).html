#Scatterplots for Important Features
import matplotlib.pyplot as plt
import seaborn as sns

# List of top features affecting Price
top_features = ['rooms', 'low_income', 'student_teacher_ratio', 'crime_rate']

# Plot scatterplots
for feature in top_features:
    plt.figure(figsize=(6,4))
    sns.scatterplot(x=df[feature], y=df['price'])
    plt.title(f'Price vs {feature}')
    plt.xlabel(feature)
    plt.ylabel('Price')
    plt.show()

Insights from EDA¶

  • Houses with more rooms tend to have higher prices (positive relationship).
  • Areas with higher low income population have lower house prices (negative relationship).
  • Higher crime rate is associated with lower house prices.
  • Student-teacher ratio also shows a slight negative impact on price.
  • The correlation matrix confirms strong relationships between some features and price.
  • Boxplots show presence of outliers in features like price and low_income.
  • Histograms indicate that some features are not normally distributed (skewed data).
