# 📊 Housing Data Exploratory Data Analysis (EDA)  
**Task 2(Level 1) – Codveda Data Analytics Internship**

--------------------------------------------------------------

## 📌 Overview  
This project focuses on performing **Exploratory Data Analysis (EDA)** on a housing dataset.  
The goal is to analyze data patterns, relationships, and key factors affecting house prices using statistical methods and visualizations.

--------------------------------------------------------------


## 🎯 Objectives  
- Understand dataset structure  
- Analyze feature distributions  
- Identify outliers and anomalies  
- Explore relationships between variables  
- Discover factors influencing house prices  

--------------------------------------------------------------


## 📂 Dataset Information  
**Dataset:** Housing dataset  

Contains features such as:  
- Crime rate  
- Number of rooms  
- Property tax rate  
- Distance to employment centers  
- Student-teacher ratio  
- Low income population percentage  
- Median house price  

--------------------------------------------------------------


## 🛠️ Tools & Technologies  
- Python 
- Pandas   
- NumPy  
- Matplotlib   
- Seaborn  

--------------------------------------------------------------


## ⚙️ EDA Workflow  

### 1. Data Loading  
- Loaded dataset using Pandas  

### 2. Data Understanding  
- Checked dataset structure, columns, and data types  

### 3. Summary Statistics  
- Used `.describe()` to analyze mean, median, and distribution  

### 4. Distribution Analysis  
- Plotted histograms for all features  
- Observed skewness in variables like crime rate and population  

### 5. Outlier Detection  
- Used boxplots to detect outliers  
- Found outliers in:
  - Crime rate  
  - Price  
  - Low income population  

### 5. Correlation Analysis  
- Created heatmap to analyze relationships  
- Identified strong positive and negative correlations  

### 6. Relationship Analysis  
- Scatter plots used to study feature relationships:
  - Price vs Rooms  
  - Price vs Income  
  - Price vs Crime rate  
  - Price vs Student-teacher ratio
